«Киберполигон» — это интерактивная браузерная игра в стиле киберпанк, где игрок формирует команду специалистов и принимает тактические решения для отражения кибератаки в условиях жёсткого таймера. Каждое действие напрямую меняет уровень системного ущерба и итоговый счёт, а детальная аналитика и таблица лидеров
